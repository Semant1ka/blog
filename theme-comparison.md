# Hugo Blog Theme Comparison

## Requirements
1. Forever free, no paywall for readers
2. Full content ownership
3. Markdown support, image assets, mermaid diagrams
4. SEO / discoverability
5. Clean & modern visual style

---

## Top 5 Theme Picks

### 1. Blowfish (Recommended)

- Built on Tailwind CSS 3, very actively maintained (2026)
- **Mermaid, Chart.js, KaTeX built-in** - no extra setup
- Light/dark mode with system preference detection
- Clean typography, multiple layout options
- Search, table of contents, image galleries
- Demo: https://blowfish.page/
- GitHub: https://github.com/nunocoracao/blowfish

### 2. PaperMod

- The most popular Hugo theme - battle-tested, huge community
- Three layout modes (regular, home-info, profile)
- Light/dark mode, auto table of contents, cover images
- Near-perfect PageSpeed scores, zero JS build tooling
- **Mermaid requires a custom shortcode** (a few lines of config - easy but not built-in)
- Demo: https://adityatelange.github.io/hugo-PaperMod/
- GitHub: https://github.com/adityatelange/hugo-PaperMod

### 3. Paper

- Ultra-distilled version of PaperMod - even more minimal
- Four curated color schemes (linen, wheat, gray, light)
- Beautiful typography, content-first design
- Great if you want the cleanest possible look
- **Mermaid requires manual setup** like PaperMod
- Demo/GitHub: https://github.com/nanxiaobei/hugo-paper

### 4. Stack

- Card-based layout, modern feel
- Built-in search, dark mode, table of contents
- Good code highlighting, clean sidebar navigation
- Active community
- **Mermaid via shortcode**
- Demo: https://demo.stack.jimmycai.com/
- GitHub: https://github.com/CaiJimmy/hugo-theme-stack

### 5. Congo

- Clean, minimalist, Tailwind-based (like Blowfish but simpler)
- Multiple homepage layouts
- Built-in mermaid, KaTeX, chart support
- Good typography, fast
- Demo: https://jpanther.github.io/congo/
- GitHub: https://github.com/jpanther/congo

## Hosting Decision: GitHub Pages

- Free forever
- Custom domain support
- Auto-deploy via GitHub Actions
- Full ownership (content lives in your git repo)

## Discoverability Strategy

- Strong SEO from static HTML + fast load times
- Cross-post to dev.to and Hashnode with canonical URLs pointing to your blog
- This ensures search engines credit your site as the original source
